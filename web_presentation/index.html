LoRa Panic Button

A rugged, low-cost (<€15) beacon for local percebes pickers.

  • Blind Spot Comms: Works under cliffs where cell phones fail.
  • Instant Alert: Sends GPS + ID directly to the mesh.
  • "Man Down": Accelerometer detection for falls.
Drift Vector API
🌊
Live Search Grids

Replacing static models with real-time sensor data.

  • Input: Real-time Current Speed, Direction & Wind.
  • Output: 60-min "Probability Cone" for Man-Overboard scenarios.
  • Value: Shrinks the search radius, saving helicopter time (€3k/hr).
Cliff Sentry
👁️
Thermal Perimeter

Automated monitoring of "No Go" zones during storms.

  • Tech: Thermal/IR cameras on coastal nodes.
  • Detection: Identifies heat signatures in restricted areas at night.
  • Prevention: Early warning before an accident occurs.
Integration Goal: All data feeds directly into a unified "Command Dashboard" for the Vila do Bispo Bombeiros.
