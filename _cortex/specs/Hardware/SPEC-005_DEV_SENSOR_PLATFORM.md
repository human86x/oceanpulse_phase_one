## 1. Overview
This specification will define the implementation of the M1 milestone: building a full sensor-to-dashboard pipeline using cheap development sensors.

## 2. Scope
- Sensor abstraction layer in Arduino firmware.
- Drivers for TDS (analog), DS18B20 (OneWire), and pH (analog).
- Serial protocol extensions for multi-sensor support.
- Dashboard updates for real-time visualization.
